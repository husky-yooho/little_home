#### 方法函数
    1. 命名以def开始，以：结束，可接受入参
    2. 参数传递：（值传递（数、字符、元组）、对象传递（列表、字典））
    3. 参数类型：
        1. 必备参数
        2. 关键参数：    多个必备参数，位置可以不一致
        3. 缺省参数：    入参的时候，就赋了个值
        4. 不定长参数：   *var_args来接收不定长的参数，放入到list中，可以遍历
        5. 匿名函数：     方法名 = lambda [入参1，入参2]：表达式；    返回分号后的内容
        6. 变量作用域     全局变量、局部变量
